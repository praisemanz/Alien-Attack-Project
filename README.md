# Alien Attack: now available on Vercel

A space shooter game where you defend Earth from waves of alien invaders. Pilot your ship, dodge enemy fire, collect power-ups, and fight through 10 increasingly difficult levels.

## Play


### Controls

| Key | Action |
|-----|--------|
| Arrow Left / Right | Move ship |
| Space / Arrow Up | Fire missile |
| P | Pause |

### Features

- **10 Levels** of increasing difficulty with unique names and alien configurations
- **4 Alien types**: Basic, Fast, Tank, and Elite — each with different HP and behavior
- **Boss fights** on levels 5, 9, and 10 with spread-shot attacks and health bars
- **Power-ups**: Shield, Rapid Fire, Triple Shot, Extra Life
- **Lives system** with heart display (up to 5)
- **Score tracking** with local high-score persistence
- **Particle effects**, screen shake, and animated starfield background
- **Mobile touch controls**


## Legacy

Originally created as a C++ / SFML project for Professor Pat Smith's Programming II class. The C++ source files (`main.cpp`, `Pixie.cpp`, `AlienArmy.cpp`, etc.) are preserved in the repo.
